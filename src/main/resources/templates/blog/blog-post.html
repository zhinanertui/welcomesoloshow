使用acme.sh定期自动更新https证书

搞个Tomcat自动配置证书

安装 acme.sh

					
curl  https://get.acme.sh | sh

生成证书

					
cd .acme.sh/
#获取你的阿里云API Key Secret:
export Ali_Key="********"
export Ali_Secret="********"
./acme.sh --issue --dns dns_ali -d example.com -d *.example.com
#生成到/home/ssl目录下
./acme.sh --install-cert -d example.com --cert-file /home/ssl/cert.pem --key-file /home/ssl/key.pem --fullchain-file /home/ssl/fullchain.pem --ca-file /home/ssl/ca.pem
